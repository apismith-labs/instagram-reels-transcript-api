# cURL examples

This directory will contain cURL examples for the Apify Actor `apple_yang/instagram-transcripts-scraper`.

Planned examples:

- Run the Actor for a single Instagram Reel URL.
- Process a CSV list of Reel URLs.
- Read `APIFY_TOKEN` from the local environment.
- Save response output to `sample-output/`.

No runnable cURL examples are included in this first-stage scaffold.
