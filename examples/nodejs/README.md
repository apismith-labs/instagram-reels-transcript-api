# Node.js examples

This directory will contain Node.js examples for the Apify Actor `apple_yang/instagram-transcripts-scraper`.

Planned examples:

- Run the Actor for a single Instagram Reel URL.
- Process `sample-data/instagram-reel-urls.csv` in batch.
- Read `APIFY_TOKEN` from the local environment.
- Save structured output to `sample-output/`.

No runnable Node.js examples are included in this first-stage scaffold.
